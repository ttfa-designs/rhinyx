# Rhinyx Website

A modern, professional website for Rhinyx - a South African mining and industrial support company.

## Overview

Rhinyx specializes in heavy equipment hire, operational support services, and integrated mining solutions for mining houses, contractors, and industrial clients.

## Features

- **Modern Design**: Dark, sophisticated theme inspired by architectural design principles
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Smooth animations and transitions
- **Multiple Pages**:
  - Home - Company overview and services
  - About Us - Company profile, mission, vision, and values
  - Services - Detailed service offerings
  - Contact - Contact information and inquiry form

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Montserrat & Roboto)
